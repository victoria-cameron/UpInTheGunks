Help

Login requests three pieces of information, username, password, and project ID. The username and password are standard validation associations. Project ID is the assignment of what actual project you are associated with. For some users associated with more than one project this is very important and allows one user to have different access levels in different projects. Here a session is created for the user that keeps track of their project, name, and role through the remainder of their time on the site.


Fill > Surveys is the site section for completing an interview. If a surveyor wants to fill a particular survey, they just have to double click on it and it opens the survey to be filled. Upon completion, they just have to click the submit button and they will see a confirmation of their addition to the records. Questions in Fill also have a small ‘+’ under them. If a surveyor gets any added information that they want to include as a part of their interview, they can use this to add a note to that answer. 


View > Records displays interviews that have already taken place. After a survey is filled, the record of it can be found here. Double clicking on a record takes you to a listing of all the questions and answers for the record you selected. There is also a handy back button on the interview to return you to the list of records.


View > Surveys displays surveys that are available to be filled. After a survey is created, you can find it here as a part of the survey list. Double clicking on a record takes you to all of the questions that are on that particular survey. While it shows you the questions, it does not let you fill the survey which you would have to go to the Fill page for.


Create > Questions allows users with access rights to add questions that can be used in the surveys. Questions can then take on three types, multiple choice, text box, and paragraph. Selecting multiple choice then allows users to populate the choice text that they want users to pull from while filling out a survey. Text box provides a simple one line space for user data to be entered. Paragraph provides a larger area for longer open ended answers.


Create > Surveys allows users to select a set of created questions and create a survey with them. Simply select the questions you wish to add with the associated checkboxes, add a title and description, and you’ll have a created survey. To see a survey that you have made, users can check for it in the View > Surveys page.


Settings provides the users with a space to make changes to their profile. If a user would like to make adjustments to their own settings, like password changes, they can do so here. Those at a higher admin level are able to create new project users and also adjust users access level for projects.
